# Single Layer Perceptron — Banknote Authentication

**CS3807 – Deep Learning Laboratory, Experiment 1**
Shiv Nadar University Chennai | B.Tech AI & Data Science, Semester V

A Single Layer Perceptron implemented completely from scratch in NumPy (no deep learning
libraries) to classify banknotes as authentic or forged, using the UCI Banknote Authentication
Dataset.

## Overview

This project covers the full pipeline end to end:

- Exploratory Data Analysis (histograms, correlation heatmap, pairplot, boxplots, class balance)
- Preprocessing (standardization, stratified 80/20 train-test split)
- A perceptron built from scratch — weight/bias init, step activation, forward pass, and the
  perceptron learning rule — with full epoch-by-epoch training logs
- Evaluation (accuracy, precision, recall, F1-score, confusion matrix)
- A learning-rate sensitivity study (0.001 / 0.01 / 0.1)
- A cross-check against `sklearn.linear_model.Perceptron`
- A 2D decision-boundary visualization
- Extra experiments: Step vs. Sigmoid activation, why a single-layer perceptron can't solve XOR
  (demonstrated experimentally), and the effect of feature normalization on convergence

## Results

| Metric    | Scratch Perceptron | Scikit-learn Perceptron |
|-----------|--------------------|--------------------------|
| Accuracy  | 98.55%             | 97.82%                   |
| Precision | 96.83%             | 96.77%                   |
| Recall    | 100.00%            | 98.36%                   |
| F1-score  | 98.39%             | 97.56%                   |

Trained for 50 epochs at a learning rate of 0.01 on 1097 training samples, evaluated on 275
held-out test samples.

## Dataset

**Banknote Authentication Dataset** — UCI Machine Learning Repository, ID 267.
Fetched programmatically via the `ucimlrepo` package, so no manual download is required.

- 1372 instances, 4 numerical features (Variance, Skewness, Curtosis, Entropy), 2 classes
- No missing values

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/<your-repo-name>.git
cd <your-repo-name>
```

### 2. Set up a virtual environment (recommended)

```bash
python3 -m venv venv
source venv/bin/activate      # Windows: venv\Scripts\activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the notebook

```bash
jupyter notebook notebook/deep_learning_lab_1.ipynb
```

or open it directly in [Google Colab](https://colab.research.google.com/) and run all cells
top to bottom. The notebook fetches the dataset itself — nothing needs to be downloaded by hand.

### 5. Or run the plain script

```bash
python src/deep_learning_lab_1_final.py
```
